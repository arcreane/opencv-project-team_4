README — Multimedia Image Processing Project
Technologies Used
Programming Language: C++
GUI Framework: Qt 6.7.3
Computer Vision Library: OpenCV
(OpenCV/opencv/build configured as global variable)
Build System: CMake / Qt Creator
Version Control: Git + GitHub
Project Folder Structure
opencv-project-team_4
│
└── opencv-project-team_4
    │
    └── application_Team4
File Organization

The project files are divided into three main categories:

Form Files → .ui
Header Files → .h
Source Files → .cpp
How to Open the Project
Open Qt Creator
Open the application_Team4 project
Configure the Qt Kit (Qt 6.7.3 recommended)
Verify that OpenCV is correctly configured
Build and run the project
OpenCV Configuration

The project uses OpenCV configured through a global variable pointing to:

OpenCV/opencv/build

Make sure OpenCV is installed correctly before building the application.
